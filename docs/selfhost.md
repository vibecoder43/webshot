# Self-host a service

## Requirements
- Linux
- The service needs delegated writable cgroups v2

## Bare NixOS

Use the module as a regular Github input. See `packaging/nixos` for reference.

## Docker and OCI

No easy way to run atm. Technically possible.

## VM image

No easy way to run atm. Technically possible.

## Official instance

The official instance runs on a bare NixOS host as a systemd service.
